# Portfolio Website Explainers

This folder contains comprehensive guides to help you customize, deploy, and maintain your portfolio website.

## 📚 Guide Overview

### 🚀 [Setup Guide](setup-guide.md)
**Start here!** Quick checklist and overview of all tasks needed to get your portfolio live.
- 30-minute quick start checklist
- File structure overview  
- Customization priorities
- Common mistakes to avoid

### 🎨 [Customization Guide](customization-guide.md)
Detailed instructions for personalizing your website with your content and branding.
- Replace placeholder content
- Update colors and fonts
- Add your photos and projects
- Customize resume section
- Create blog posts

### ☁️ [Azure Deployment Guide](azure-deployment-guide.md)
Step-by-step instructions for publishing your website to Azure Static Web Apps.
- GitHub repository setup
- Azure configuration
- Custom domain setup
- Automatic deployment workflow
- Cost management

### 🔧 [Troubleshooting Guide](troubleshooting.md)
Solutions for common issues you might encounter.
- Development problems
- Deployment issues
- Responsive design fixes
- Performance optimization
- Emergency recovery

### 📝 [Blog Content Examples](blog-content-examples.md)
Templates and ideas for creating engaging product design blog posts.
- Blog post templates
- Content structure examples
- Topic ideas for designers
- Visual content guidelines
- Publishing checklist

## 🎯 Quick Start Path

**New to web development?** Follow this order:
1. Read [Setup Guide](setup-guide.md) - Get oriented
2. Follow [Customization Guide](customization-guide.md) - Make it yours
3. Use [Azure Deployment Guide](azure-deployment-guide.md) - Publish online
4. Keep [Troubleshooting Guide](troubleshooting.md) handy - Fix issues
5. Use [Blog Content Examples](blog-content-examples.md) - Start writing

**Experienced developer?** Skip to:
- [Customization Guide](customization-guide.md) for content updates
- [Azure Deployment Guide](azure-deployment-guide.md) for hosting

## 🆘 Getting Help

**Stuck on something?**
1. Check the relevant guide in this folder
2. Look for error messages in browser console (F12)
3. Search the specific error message online
4. Try the solutions in [Troubleshooting Guide](troubleshooting.md)

**Still need help?**
- Stack Overflow for technical questions
- Azure documentation for hosting issues
- GitHub community for repository problems
